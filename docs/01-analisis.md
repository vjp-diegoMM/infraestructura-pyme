# Análisis de Requisitos

## Cliente
Pequeña empresa con necesidad de presencia web y gestión interna.

## Requisitos funcionales
- Servidor web con PHP para alojar la web corporativa
- Base de datos MySQL para la web y otra para gestión interna
- Acceso remoto seguro mediante SSH
- Monitorización del sistema en tiempo real
- Copias de seguridad automáticas con rotación

## Requisitos no funcionales
- Sistema operativo: Ubuntu Server 22.04 LTS
- Alta disponibilidad básica
- Recuperación ante fallos en menos de 4 horas

## Restricciones
- Presupuesto limitado: soluciones open source
- Un único servidor físico en fase inicial
